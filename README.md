# Chaste iView Admin

简洁的基于 Vue & iView 的后台管理系统模板

## Getting started
```bush
# clone the project
git clone https://github.com/chastephp/chaste-iview-admin.git

// install dependencies
npm install

// develop
npm run serve
```

## Build
```bush
npm run build
```
